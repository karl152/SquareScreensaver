# SquareScreensaver
a screensaver with squares<br>
<img src="https://raw.githubusercontent.com/karl152/SquareScreensaver/refs/heads/main/icon.png" alt="SquaredScreensaver Icon"><br>
**IMPORTANT**: This "Screensaver" focuses more on providing a customizable animation than actually "saving the screen" (preventing CRT burn-in).
### How to install
* download the ZIP archive from the releases section
* extract the ZIP archive
* run `install.cmd` as Administrator
#### How to uninstall
* clear the configuration in the screensaver settings if you want
* select another screensaver
* uninstall it like any other Windows software
  * Settings -> Apps -> Installed Apps -> SquareScreensaver -> Uninstall
  * *or* Control Panel -> Uninstall programs -> SquareScreensaver -> Uninstall/Change
### Compatibility
SquareScreensaver is compatible with Microsoft Windows NT 10 (modern Windows 10 and Windows 11)
